# Проект Mesto.
## Вторая проектная работа по адаптивно-отзывчивой верстке.

Веб-приложение с возможностью создать свой профиль, делиться фотографиями и ставить классы.


Проект реализован с помощью:

1. HTML
2. CSS
3. Flex-box позиционировани
4. Grid
5. БЭМ - методологии
6. Javascript
